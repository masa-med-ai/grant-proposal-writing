# 32. 基盤研究C 記載注意事項（評定要素 & 書き方ルール）

> Step3で**必ず最初に読み込む**ファイル。若手研究より評定項目が1つ多いことに注意（着想経緯・研究動向）。

## 1. 種目の概要

- 名称：科学研究費助成事業「基盤研究(C)」（基盤Bも採点要素は同一構造）
- 対象：1〜数人の研究者で行う比較的小規模な研究
- 期間：3〜5 年
- 直接経費：500万円以下

## 2. 評定要素（**(1) 内の・項目それぞれが独立採点される**）

### (1) 研究課題の学術的重要性

- **A.** 学術的に見て、推進すべき重要な研究課題であるか
- **B.** 研究課題の核心をなす学術的「問い」は明確であり、学術的独自性や創造性が認められるか
- **G.** 着想に至った経緯や、関連する国内外の研究動向と本研究の位置づけは明確であるか

  > **若手研究との最大の違い**：G が独立採点項目として加わる。
  > G は更に **G1: 着想に至った経緯** と **G2: 国内外の研究動向と本研究の位置づけ** に分けて見られる。

### (2) 研究方法の妥当性

- **C.** 研究目的を達成するため、研究方法・研究計画は具体的かつ適切に検討されているか
- **D.** 研究目的、研究方法に妥当性、独自性、創造性は認められるか

### (3) 研究遂行能力及び研究環境の適切性

- **E.** これまでの研究活動、研究実績等から、研究計画の遂行が可能であると判断できるか
- **F.** 研究環境（研究体制、研究施設・設備等）は研究の遂行に十分であるか

> A〜G がそれぞれ独立採点。**G を専用の小見出しで立てることが基盤Cでは必須**。

## 3. 本文の必須小見出し

「研究目的」セクション内に必ず以下の小見出し（##）を立てる：

- ## 本研究の核心をなす学術的「問い」（評定B）
- ## 学術的独自性と創造性（評定B）
- ## 着想に至った経緯（評定G1）
- ## 国内外の研究動向と本研究の位置づけ（評定G2）
- ## 本研究で何をどこまで明らかにしようとするのか
- ## 本研究の学術的重要性（評定A）

「研究方法」セクション内に必ず以下の小見出し（##）を立てる：

- ## 全体研究計画の概要
- ## 各研究項目（WP）の詳細（評定C）
- ## 研究方法の妥当性・独自性・創造性（評定D）
- ## 想定されるリスクと対応策

## 4. ページ・文字数の制約

- 様式：Wordで **4ページ以内**（研究目的＋研究方法）
- 1ページ ≒ 1,200〜1,400字
- 図表は最低1枚配置を推奨

## 5. G（着想経緯 & 研究動向）の書き方ガイド

### G1: 着想に至った経緯
- 申請者の **これまでの研究** からの自然な発展であることを示す
- 予備データ／予備実験／既発表論文 への参照
- 「○○の臨床現場で××という未解決課題に直面した」のような具体的エピソードが効果的

### G2: 国内外の研究動向と本研究の位置づけ
- **最新3年以内**の主要論文を国内・海外ともに引用
- 競合する研究グループを **正確に同定**し、本研究との差別化点を明示
- レビュー論文を引用しつつ、未解決領域（gap）を1段落で抽出
- 引用フォーマット：1st author / 雑誌 / 年 + ハイパーリンク

## 5-A. 平易化ルール（門外漢の専門家が読むことを前提に）

基盤Cの審査員は **博士号レベルだが、必ずしも同じ専門分野ではない** 研究者です。
特に G1（着想経緯）と G2（研究動向と位置づけ）は **他分野の審査員にも筋道が伝わるか** が決め手になります。

### 守るべき4原則

1. **専門用語には必ず短い言い換えを括弧書きで添える**
   - NG：「○○病変を△△的に切除し…」
   - OK：「○○病変（前がん病変のひとつ）を△△的に切除し…」
2. **略語・英略語は初出でフルスペル＋日本語訳**
   - NG：「AIによる××を用いて…」
   - OK：「人工知能（AI; Artificial Intelligence）を用いた××（フルスペル；日本語訳）により…」
3. **カタカナ語は1段落あたり概ね3語以内**
   - 日本語に置き換えられる場合は置き換える（例：「ワークフロー」→「作業の流れ」）
4. **修飾の入れ子をやめる**
   - NG：「○○ベースの△△機構を組み込んだ××型モデル」
   - OK：「○○を自動で抽出する△△モデル（××型）」

### NG表現と推奨表現の対応表（例）

| NG（密度が高すぎる） | 推奨 |
|------------------|------|
| エビデンスベースで | 科学的根拠に基づき |
| パラダイムシフト | 大きな転換 |
| ロバストな | 安定して動く |
| インプリメンテーション | 実装（社会への組み込み） |
| プラットフォーム | 基盤システム |
| バイアス | 偏り |
| アウトカム | 結果指標 |

> 少数の英単語は許容しますが、文章のリズムが英語に支配されると **「内向きの専門家文書」** と映り、評定が下がります。

## 5-B. 著者本人の論文の自然な織り込み方（自己引用ポリシー）

- **Step1-5 で確定した自己引用候補**（`10_Step1_テーマ壁打ち/13_テーマ候補メモ.md` 内）から、**1〜3本** を本文に織り込む。
- 推奨セクションは以下のいずれか（**基盤Cでは特に G1 で活きる**）：
  - **「着想に至った経緯」（評定G1）**：「申請者は先行研究 ◯◯ら, 雑誌名, 年 において△△を示した。これにより未解決の問いとして××が浮上した。」
  - **「予備データ／申請者の研究実績」**：「申請者の先行研究（◯◯ら, 雑誌名, 年）では N=◯◯ において…」
  - **「研究方法の妥当性」（評定D）**：「本手法は申請者が既報（◯◯ら, 雑誌名, 年）で確立した解析パイプラインを発展させたものである。」
- **NG**：「申請者は世界初の…」のような自慢調・列挙調。あくまで先行研究として淡々と参照する。
- **NG**：自己引用ばかりで他者研究の引用がない。**国内外の他者論文との比率は1:3 程度** を目安に。
- 引用フォーマット：1st author / 雑誌名 / 発刊年 + PubMedハイパーリンク。

## 6. 執筆時の自己チェックリスト

- [ ] A 学術的重要性に対応する段落がある
- [ ] B 学術的「問い」が小見出しで1文明確
- [ ] B 独自性・創造性が小見出しで明確
- [ ] **G1 着想経緯が独立小見出しで書かれている**
- [ ] **G2 国内外研究動向と位置づけが独立小見出しで書かれている**
- [ ] C 研究方法が WP / MS で具体化
- [ ] D 方法の妥当性・独自性・創造性が触れられている
- [ ] E 申請者実績への言及
- [ ] F 研究環境への言及
- [ ] 図表1枚以上
- [ ] 4ページに収まる試算
- [ ] **平易化**：専門用語に短い言い換えが付いている／カタカナ語が密集していない
- [ ] **略語**：初出すべてにフルスペル＋日本語訳が併記されている
- [ ] **自己引用**：`13_テーマ候補メモ.md` の自己引用候補から最低1本を本文に織り込んでいる（特に G1 で活用）
- [ ] **引用バランス**：自己引用と他者引用の比率がおおむね 1:3 以内
