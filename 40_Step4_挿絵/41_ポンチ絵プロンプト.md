## ポンチ絵の基本方針

科研費調書のポンチ絵は以下を満たすと評価が高い：

- 一目で **「問い → 方法 → 期待される成果」** が読み取れる
- 左→右、上→下の **時系列フロー**が明確
- 数値マイルストーン（N=500、AUC≥0.90 等）が図中に書かれている
- 配色は2〜3色に抑える（白背景、メインカラー1色、アクセント1色）
- 文字は **日本語**（査読官が読むので英語不可）。フォントはゴシック系。

### ChatGPT (DALL·E / GPT-image) 用プロンプト

```
A clean, professional infographic for a Japanese scientific research grant application (KAKENHI).
Style: minimalist, flat design, 2-3 color palette (navy blue, light gray, accent orange), white background, no photographic elements.
Layout: horizontal 16:9, clear left-to-right flow.

Left panel — "Research Question (学術的問い)":
[ここに本研究の「問い」を1〜2行で英語で記述。後で日本語に差し替え可]

Center panel — "Methods (研究方法)" with three connected boxes (WP1 → WP2 → WP3):
- WP1: [WP1のキーワード]
- WP2: [WP2のキーワード]
- WP3: [WP3のキーワード]
Use arrows to show pipeline.

Right panel — "Expected Outcomes (期待される成果)":
- Quantitative milestone 1: [例: N=500 prospective cohort]
- Quantitative milestone 2: [例: AUC≥0.90]
- Quantitative milestone 3: [例: 3 papers in IF>10 journals]

Constraints:
- All Japanese text rendered in clean Gothic font.
- No human faces, no medical illustrations of organs, abstract icons only.
- Leave 5% margin on all sides.
- Output: 1920x1080 PNG.
```

## 生成後の確認チェックリスト

- [ ] 「問い」が図中に書かれている
- [ ] 数値マイルストーンが少なくとも2つ書かれている
- [ ] 矢印または番号で時系列・因果が読み取れる
- [ ] 配色が3色以下
- [ ] 日本語フォントが崩れていない（生成画像でガビガビになっていない）
- [ ] 印刷時にA4半分のサイズで読めるか（解像度確認）

## 図差し替え案

生成画像のテキストが崩れた場合は、画像はベース構図のみ生成し、**最終的にPowerPoint/Keynoteでテキストを上書き**することを推奨。
